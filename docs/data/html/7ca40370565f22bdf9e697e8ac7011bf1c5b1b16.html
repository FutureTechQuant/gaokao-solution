活动介绍

为了更好的服务2012年高考考生，帮助广大考生在志愿填报时能够了解高校专业情况，科学合理填报志愿，中国教育在线高考频道特举办2012年“零距离看专业”大型在线咨询活动。热门专业在校大学生志愿者，就自己所就读专业的开设课程、专业介绍、就业前景、职业生涯规划等相关问题进行在线解答，从过来人的角度，给予客观、公正的指导，为考生选择专业提供参考。“零距离看专业”活动将拉近考生选专业的距离，让高考生了解自己更加适合什么专业！

“零距离看专业”活动将拉近考生选专业的距离，让考生了解自己更加适合什么专业！选择学校、选择专业的方法有很多....

有人这样选学校——重点大学、大城市、学费低、离家近；可以保研、学校学风好……

有人这样选专业——自己感兴趣、就业机会多、专业发展空间大；考研容易、将来工作稳定、未来薪水高……

象牙塔内的您或是工作中的您，当初是如何做出自己的选择？您可以加入“零距离看专业”活动的志愿者，也可以写出您建议给我们投稿！

大学生志愿者招募

招募范围：

热心志愿服务、愿意为考生提供报考指导的在校大学生及对选择专业有经验的人士！

招募条件：

（1）各高校在校大学生；

（2）对所学专业基本情况、课程设置、发展前景、历年的招生录取情况、就业情况有深入认识；

（3）对大学生职业生涯规划的相关知识有所了解；

（4）具备上网条件，并有一定的固定上网时间；

（5）有较强责任心、认真负责、热心志愿服务。

服务形式：

网上回答考生专业选择相关问题

服务时间：

活动期间（2012年4月—8月）

申请办法：

点击申请　零距离看专业志愿者(http://ask.eol.cn/volunteerReg.htm)

第一步：注册eol通行证（如果有通行证可以直接登陆申请志愿者）；

第二步：填写志愿者真实信息，包括姓名、学校、专业类别、专业、邮政编码、联系地址、联系电话等；

奖项设置

一等奖1名：320G移动硬盘

二等奖2名：16G优盘

三等奖6名：品牌鼠标

优秀志愿者100名：颁发中国教育网、中国教育在线2012年“零距离看专业”优秀志愿者荣誉证书。

评选标准：

1、考生、考生家长等网友对问题答案满意度；

2、志愿者回答问题的及时性和准确程度；

3、志愿者所在讨论区网友问题的解答比例。

欢迎申请中国教育在线2012年“零距离看专业”活动志愿者！
