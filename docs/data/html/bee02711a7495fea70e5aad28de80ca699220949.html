中国教育在线2004年度广告授权经营管理办法

1、授权经营体系

1．1中国教育在线网络广告授权经营体系由北京赛尔网络有限公司（以下简称“赛尔”）信息服务事业部广告销售部（以下简称“广告销售部”）、中国教育在线网络广告销售授权经营商（以下简称“经营商”）组成。

1．2营商必须具备的条件：

(a)具有独立法人资格；

(b)有权从事广告经营业务。

2、经营商发展

2．1申请：申请单位向网络销售部提出书面申请，填报《中国教育在线广告销售授权经营申请登记表》并提交相关资料，包括：

a)申请单位法定代表人身份证复印件、股东名单；

b)申请单位营业执照及广告经营许可证复印件；

c)申请单位税务登记证复印件。

2．2审核：广告销售部对申请单位提供资料进行审核。

2．3授权：双方协商一致后签署中国教育在线网络广告销售授权经营协议。

3、有效期限及适用范围

3．1管理办法自2004年8月1日起执行，有效期至2004年12月31日止。（即：经营商完成广告销售任务的额度自2004年8月1日起统计）；

3．2本办法非年初开始执行，框架协议按自然年度计算，所以依据本办法的年底返点按框架协议细分到月的金额结合签约日期综合计算；即经营商按最低完成月平均额度*签约月度，经营商完成月度平均，即可享受返点；

4、解释权与裁定权

4．1管理办法最终解释权与裁定权归北京赛尔网络有限公司所有。

4．2北京赛尔网络有限公司有权随时根据需要修改本管理办法。
