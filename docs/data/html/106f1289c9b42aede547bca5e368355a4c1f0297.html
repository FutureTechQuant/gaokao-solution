澳门地区高校内地招生条件

高考成绩须过一本线

　　澳门高校录取原则上为择优录取，考生高考成绩一般需达到其所属省的本科第一批次录取分数线，个别院校参考本科第二批次录取分数线。

英语单科成绩超110分

　　澳门高校内地招生英语成绩要求高，英语单科分数如果满分150分的话，澳门高校则要求要生的英语成绩在110分以上。

特色专业

　　澳门本身是一个旅游、娱乐业发达的城市，博彩业管理旨在为博彩业界有志进修的在职人员，或有志在澳门博彩娱乐业发展的高中应届毕业生开设。

学历认可

　　澳门6所院校可在内地25省（市）招生，学生在澳门学业期满，获得高等学校颁发的学历、学位证书，内地教育行政部门予以承认。

就业机会

　　调查结果显示，2013年1月至3月澳门失业率为1.9%，维持在历史最低水平。

澳门饮食

　　澳门烹饪吸收了广东地区的烹饪法和食材，以及香港、萄葡牙、印度、非洲、东南亚的特色。创制出独一无二的澳门菜。例如烧腊、多士、非洲鸡、马介休、葡国鸡、沙嗲等。澳门人的饮食习惯也是一日三餐。早餐时间，一般在早上7点。早餐内容各人自有差别。一顿简单的早饭，可能为方便面、白粥、肠粉、车仔面、面包以及咖啡奶茶。澳门也有下午茶的时间，一般在下午4时半左右，多以车仔面、多士、猪扒饱咖啡奶茶为主。

生活习惯

　　澳门，一个经济飞速发展的城市，据统计其人均GDP大体上已经与日本旗鼓相当。不过这个以赌业及旅游业支撑的弹丸之地尽管已经取得巨大的进步仍然难掩其经济上的畸形和收入水平生活水平上的巨大差距，不如内地城市。近些年来澳门也新建了一些豪华住宅，但遍布于市中心的仍旧是破败不堪的老旧建筑。以及昏暗狭窄的陋巷。如果不是亲眼所见难以想象人均GDP高居亚洲最前列的澳门竟然如此的不堪。

消费水平

　　亚洲开发银行最新公布的调查报吿显示，澳门生活水平已跻身亚洲23个国家和地区的第五位，仅次香港、台湾、文莱和新加坡。澳门开支最大是住。除了租房子比内地高出好几倍，餐厅吃的贵一、两倍，住酒店贵两至四倍，其他都不算高，八百半百货、超市里好多商品和内地差不多，有的还比内地便宜，奢侈品、电器、手机、电子产品、化妆品、名牌时装、牌子货比内地便宜很多，大家乐、麦当劳等，快餐价钱基本和内地相同。
