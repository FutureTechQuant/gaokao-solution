  中国教育在线大事记
 
 

1）EOL.CN(中国教育在线)拥有的品牌优势：

中国教育在线作为教育部的信息发布平台，拥有最权威、最丰富、最及时的独家教育信息。同时历经4年在高校市场和教育市场中的培育，中国教育在线在高校在校和毕业生中拥有广泛的有户群体，目前这个群体已经扩展到高中生和社会上有教育以及在教育需求的广大人群。

2）中国教育在线的网民特征：

a）网民分布地域和行业广泛

b）网民平均年龄22岁，其中具消费（潜）力的占72%

c）7成网民拥有大专以上学历

d）55.7%的网民是（高校）学生

e）32%的网民是高校教师、IT行业和政府机关部门人员

f）7%的网民是各类公司管理层人员

3)频道优势：

1）中国教育在线高考频道—高中生获取高招信息和高招咨询的最佳渠道

中国教育在线的核心频道之一，拥有最全面最及时的高招信息以及线上咨询，在2004年高考招生咨询月活动中，日均访问量最高达到2600万；线上咨询形式得到了一致认可。高校招生办的访谈给广告考生解答各类问题。高考栏目使得中国教育在线成为除大学生以外的高中生及家长的认可媒体；

2）中国教育在线考研频道—最全、最新的考研信息发布频道

该频道及时发布全国200多所正规院校的研究生招生信息以及考研调剂信息，同时提供各类考研辅导信息发布，在各高校师生中拥有广泛影响。同时也是社会人员查询考研信息的最佳选择。

3）中国教育在线留学频道—最广泛国家地区留学全程信息服务以及优秀留学中介的展示平台

中国教育在线和留学中介的管理机构：涉外信息监管网拥有良好的合作关系，在这个信息发布平台上为广大有留学需求的人群提供权威信息，所发布和介绍的留学中介机构信息均为经涉外监管网批准的中介机构，使访问者在得到全面信息的同时得到真实可靠的信息。

4）中国教育在线行考频道—行业考试最及时信息以及行业考试培训最全面信息发布平台

涵盖了专业类、IT认证类、财会金融业、文秘语言医药类、经济管理类几乎全部的行业考试信息，同时通过培训信息、辅导资料、问答专区和试题专区的设置，把行考频道打造成了信息最全面、形式最丰富的行考信息平台；

5）中国教育在线教师频道—高校优秀教师招聘的保证

在这里有近百所高校找到了合适的教师人才，这里已经成为全国高校人才流动的权威招聘信息场所；全年度全方位的信息发布服务，给有志于高等教育的专业人员提供了更好的就业信息获取渠道。高校和教师在这里实现了最充分的沟通。

6）中国教育在线教育产品频道—为教育行业提供产品的服务商的最佳展示场所

拥有CISCO、华为、星网锐捷、神州数码、浪潮、INTEL、SUN、华夏科技、华文科教等忠实合作客户，中国教育在线通过广告、建立专栏和利用活动进行推广的方式给这些客户树立了良好的市场形象，给客户带来了现时的和潜在的商业价值。给教育行业内人士提供了一个选择IT产品的优良信息平台。

7）中国教育在线IT频道—提供全面IT行业和培训信息

专业的培训信息发布频道，针对广大的校园用户群体发布各类培训信息，专业有效的对培训提供方和培训需求方搭建了信息沟通的桥梁。

8）中国教育在线校园频道—拥有最广泛学生群体的时尚频道

一个针对校园的全方位立体服务提供平台，迎合时尚一族开发各类最新网络服务产品，必将成为一个吸引更多校园群体的强大磁场。
