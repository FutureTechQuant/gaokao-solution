状元未必获录取

　　状元只会提升学校知名度，让学校有面子，但学校的招生决策以学生为本，如果该考生来香港读书不利于他的长远发展，也不会被录取。

面试也要用英语

　　全英文面试考察的不仅仅是学生的英语能力，还考察一个学生的眼界是否宽广、能否很好地与别人沟通、倾听别人的意见等。

香港饮食

　　香港饮食文化为东方文化及西方文化的交汇所在，发展出了一个揉合中国菜（主要为粤菜）和西餐的饮食习惯，因而被誉为“美食天堂”。作为全球各地人们的汇聚点，日、韩、台湾、越南、泰、印度等餐厅于香港均十分常见。但亦有不少外国人都指，虽然在香港可以品尝到世界各地的美食，这些在美食其实都已经为了迁就香港人的饮食习惯而变了质，从而变得不够地道。

生活习惯

　　香港人虽然爱吃，却懒得花费精力。许多“家庭主妇”、“家庭主男”为一日三餐忙得团团转的时候，香港人却活得潇洒极了。“上班族”往往在去公司的路上，顺便买一只面包或者一块蛋糕，外加一杯牛奶，一顿早饭就这么轻轻松松地打发掉了。说到住，香港人就有点无精打采了，那是他们的“心病”。香港人的家，很少向外人“开放”，即使是很好的朋友。

消费水平

　　今天的香港，是个十分富裕的社会。台湾物价不低的，当然没有比北京、上海来得高。另外就是，不少大陆人都说：大陆贵的台湾便宜，台湾便宜的在大陆却贵。同样餐点在台湾比大陆贵上蛮多的，但是同样的化妆品却比大陆便宜很多。而这又是个先敬罗衣后敬人的社会，因而服装消费、鞋帽消费、化妆品消费量大质高。科技的进步、素质的提高，又造就了电子产品和旅游业的兴旺发达。
