　　山东省2018年普通高校招生专科(高职)提前批录取自8月5日开始，于8月6日顺利结束，本批次包括直招士官生招生高校及专业、飞行技术专科专业，共录取2844人。其中，文科814人，理科2030人。

　　专科(高职)普通批包括专科(高职)院校招收建档立卡贫困家庭学生专项计划(以下简称高职院校专项计划)和未列入专科(高职)提前批的文理类、艺术类、体育类、春季高考的专科(高职)高校及专业。高职院校专项计划录取工作已于8月7日结束，共报名207人，录取205人。

　　专科(高职)普通批其他类别首次志愿于8月8日投档，共投出考生235212人。其中，文理类共投出考生190313人(文科100826人，理科89487人)，艺术类共投出考生12813人(艺术文10824人，艺术理1989人);体育类投出考生426人;春季高考投出考生31660人。

　　专科(高职)普通批首次志愿录取结束后，未被录取的考生将有一次填报征集志愿的机会：8月11日9:00至17:00为考生填报专科(高职)普通批征集志愿时间。征集志愿计划将于填报志愿之前公布，提醒广大考生根据自身情况按时填报。
