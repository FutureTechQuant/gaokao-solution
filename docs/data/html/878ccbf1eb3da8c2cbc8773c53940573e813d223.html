2009年中国大学校园开放日时间安排

    导言：随着一年一度的高考临近，许多考生在紧张复习、备考的同时，对究竟要报考哪所大学仍是犹豫未决。自己向往的大学到底是什么样子？热爱的专业学习是怎样进行的？现在大家将有机会一探究竟。
    为了让考生充分了解学校2009年本科招生政策，给考生提供一个与学校各学院老师面对面交流的机会，自2009年4月起，将有不少高校大门陆续向考生和家长敞开，组织2009年高考招生咨询校园开放日活动。考生借此机会不仅可以更有针对性地提出自己的问题和疑惑，还可感受各大学校的校园气氛，一览各校美景。
北京 天津 上海 浙江 陕西 持续更新中,请继续关注!!
